# 柔道整復師 試験プロフィール

この文書は、柔道整復師過去問の `01〜04prompt` 作業で共通に参照する前提情報をまとめたものである。

## 試験全体の見取り図

- 対象は、柔道整復師国家試験の過去問である。
- `output/judoseifukushi/questions_json` には、1993〜2026年の34年分、合計7,600問がある。
- 1993〜2004年は各200問、2005〜2019年は各230問、2020〜2026年は各250問である。
- `00_source` の `questionType` は全問 `true_false` である。
- `questionIntent` は `select_correct` が多数派だが、`select_incorrect` も一定数あるため、設問末尾の「誤っている」「不適切」「正しくない」を必ず確認する。

## 科目構成

`output/judoseifukushi/category/category.json` は、柔道整復研修試験財団の公開資料を正本として、試験科目を `folders`、大項目を `questionSets` にしたものである。

主な領域は次の通り。

- 必修問題
- 解剖学
- 生理学
- 運動学
- 病理学概論
- 衛生学・公衆衛生学
- 一般臨床医学
- 外科学概論
- 整形外科学
- リハビリテーション医学
- 柔道整復理論
- 関係法規

## 01〜04作業への示唆

- 01 `questionType`: 現在は全問 `true_false` だが、選択肢構造と問題文を必ず確認する。
- 02 `questionIntent`: 「正しいもの」「誤っているもの」「適切でないもの」を見落とさない。
- 03 `explanationText`: 各選択肢について、正誤と理由を分けて書く。
- 04 `questionSetId`: 年度や元サイトの分類ではなく、公式出題基準のどの学習単元へ戻るかで決める。
