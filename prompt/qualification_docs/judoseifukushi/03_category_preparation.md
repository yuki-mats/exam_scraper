# 柔道整復師 `category.json` 準備メモ

この文書は、柔道整復師過去問の `questionSetId` 紐付けを、公式出題基準に沿った復習単元として保つための補助資料である。

## 分類の正本

- 共通方針は [category_taxonomy_policy.md](/Users/yuki/development/exam_scraper/prompt/qualification_docs/category_taxonomy_policy.md) を正本とする。
- 柔道整復師固有の分類は、柔道整復研修試験財団の公開資料を正本とする。
- `output/judoseifukushi/category/category.json` は、試験科目を `folders`、大項目を `questionSets` にしたものである。

## 基本原則

- `questionSetId` は、元サイトの `category` が何かでは決めない。
- 受験者がその設問を復習するとき、公式出題基準のどの学習単元へ戻るべきかで決める。
- `questionSetId` に使ってよいのは、`category.json` の `questionSets[].questionSetId` のみである。
- `folders[].folderId` を設問へ直接付与しない。

## 科目の境界

- 必修問題は、患者の権利、医療過誤、柔道整復師の基礎的理解など、全体横断の必修論点を受ける。
- 解剖学・生理学・病理学概論は、それぞれ構造・正常機能・病因や病態の基礎を受ける。
- 一般臨床医学は、症候、診察、検査、治療法などの総論を受ける。
- 外科学概論と整形外科学は、外科一般と運動器疾患のどちらが主題かで分ける。
- リハビリテーション医学は、評価、ADL、生活指導、訓練計画が主題のときに優先する。
- 柔道整復理論は、施術理論、固定、整復、リスク管理、関連学説を受ける。
- 関係法規は、免許、業務範囲、広告、罰則、届出などの法令論点を受ける。

## 迷いやすい論点

- 症候そのものの一般論は一般臨床医学、症候に対する施術理論は柔道整復理論へ寄せる。
- 法令・制度・罰則は関係法規へ寄せ、医療制度や社会保障の一般論は一般臨床医学や必修問題の主題に応じて分ける。
- 運動器疾患そのものは整形外科学、リハビリの評価・生活期対応はリハビリテーション医学へ寄せる。
- 出題当時の制度と現行制度が異なる場合は、過去問としての正答と現在の学習上の理解を分けて説明する。

## 実務上の指針

- まず `questionBodyText` で単元を仮置きし、必要なときだけ `choiceTextList` と `explanationText` を見る。
- 類似の questionSet が複数ある場合は、最も具体的で受験者の復習先として妥当なものを選ぶ。
- 「融合」「その他」に逃がすのは、主題が一つに絞れない場合だけにする。
- 関係法規は、まず `judoseifukushi_qs12_01` 〜 `judoseifukushi_qs12_07` を優先する。柔道整復師法、医療法、医師法、保健師助産師看護師法、診療放射線技師法、理学療法士及び作業療法士法、薬剤師法は、それぞれの単独項目へ寄せる。
- 公衆衛生や衛生制度は、`judoseifukushi_qs06_01` 〜 `judoseifukushi_qs06_05` を基準にし、感染症・消毒・環境衛生などの主題で判断する。
