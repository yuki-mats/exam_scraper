# 二級建築士 `lawReferences` 対象法令スコープ

この文書は、二級建築士の法規問題で `lawReferences` を作る前に参照する対象法令スコープである。

作業者は、まずこの表の範囲から法令を確認する。e-Gov の全法令から無差別に探してはいけない。スコープ外の法令を使う必要がある場合は、問題文・設問文・選択肢・解説候補に直接関係する根拠を確認し、この文書へ追記してから使う。

## 告示・条例・通達

告示・条例・通達は、問題文・選択肢・解説候補で具体名または明確な根拠がある場合だけ使う。

- `告示` という短縮表記だけで `verified` にしない。
- 具体的な告示名、告示番号、条項・項目、または一次情報で確認できる識別子を記録する。
- 条文本文を確認できない場合は `candidate` / `unverified` にとどめ、upload 前に repair する。

## スコープ外法令を追加する条件

次を全て満たす場合だけ、このスコープ外の法令を追加できる。

- 問題文・設問文・選択肢・解説候補のいずれかに、その法令が正誤根拠になる合理的根拠がある。
- 一次情報で正式法令名、`lawId`、条番号を確認している。
- この文書の `conditional` 表へ追記し、使う場面を明記している。

満たせない場合は、推測で `lawReferences` に入れない。

## 目視確認

1問ずつの `lawReferences` 目視確認では、次を照合する。

- 問題文・設問文が問う法令範囲
- 各選択肢の正誤理由
- `explanationText` の説明
- 法令文書本文の対象・要件・例外・数値
- `lawReferences` の `lawTitle` / `lawId` / `article` / `paragraph` / `item`

`lawId` が付いているだけでは合格にしない。
