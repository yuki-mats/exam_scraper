# 給水装置工事主任技術者試験 解説文戦略

この文書は、`03_prompt_add_explanationText.md` で給水装置工事主任技術者試験の `explanationText` を書く際に、何をプラスアルファとして補うべきかを整理した補助資料である。

## この試験の解説で最優先にすべきこと
- 単に `正しい。` `間違い。` と書かず、**どの判断軸でそう言えるのか**を示す。
- 特に次のどれで判定したかを明示すると学習効果が高い。
  - 主語
  - 適用範囲
  - 数値基準
  - 工程順序
  - 設備の役割
  - 協議相手・管理主体

## 頻出のひっかけ
- `水道事業者` と `都道府県知事` など主語のすり替え
- `指定給水装置工事事業者` と `主任技術者` の責務の混同
- `道路管理者`、`警察署長`、`埋設物管理者` の役割の混同
- `性能基準` と `システム基準` の混同
- `受水槽式` と `直結式` の混同
- `数値基準` の入替え
- `先に行う工程` と `後に行う工程` の逆転

## 正誤の組合せ型での解説方針
- この試験では、`questionBodyText` にア〜エ本文が並び、`choiceTextList` が `ア：正 イ：誤 ...` の組合せだけになる問題が頻出する。
- この型では、受験者が知りたいのは「どの組合せが正しいか」ではなく、「ア〜エ本文のどこが誤りか」である。
- したがって、誤答肢の解説は組合せ差分だけで終わらせず、**ズレた記号に対応する本文の誤り箇所**を必ず書く。
  - 悪い例: `この選択肢ではエが不一致である。`
  - 良い例: `この選択肢ではエを正としているが、エ本文の『任意の位置』が誤りであり、正しくは水道事業者が指示する位置に設置する。`
- 誤答肢では、差分要素中心で説明する。
  - 一致している記号は原則として主役にしない。
  - ズレた記号が1つなら、その本文のどの語句・数値・条件が違うかをピンポイントで書く。
  - ズレた記号が複数あるときだけ、その差分を順に書く。
- 正答肢でも、単に `ア：誤 イ：正 ... が正解` と書かない。
  - 誤りの記号は、本文のどこが誤りかを示す。
  - 正しい記号は、成立理由か判定軸を簡潔に添える。
- この試験では特に、次のズレを優先して言語化すると学習効果が高い。
  - 数値基準の入替え
  - 設置位置や取付条件の取り違え
  - 施工順序や確認順序の逆転
  - 用語定義の取り違え
  - 主体や管理者の混同

## 解説の基本形
- 1文目:
  - `正しい。` または `間違い。`
- 2文目:
  - 誤りの箇所、または成立理由を一言で述べる
- 3文目以降:
  - 類題でも使える補足知識を入れる

例:
```json
"間違い。\n\n誤っているのは、供給規程を定める主体を都道府県知事としている点である。供給規程を定めるのは水道事業者であり、この分野では主語の入替えが典型的な誤答肢になる。"
```

## 章別の補足知識
### 公衆衛生概論
- 原因物質と健康影響を対にして書く
- 浄水処理の目的と工程を補う
- 水質基準は「何のための基準か」を示す

### 水道行政
- 誰が定めるか、誰が届け出るか、誰が認可するかを書く
- `供給規程`、`指定事業者`、`主任技術者` の役割の違いを整理する

### 給水装置工事法
- どこに設置するか、どの順で施工するか、何を確認するかを書く
- 埋設・明示・メーター・分岐で、施工位置や確認事項を具体化する

### 給水装置の構造及び性能
- どの基準の話かを最初に明示する
- 性能基準名と、その基準が防ぎたいリスクを対応させる
- `システム基準` は、個別器具ではなく配管系統全体の安全確保であることを書く

### 給水装置計画論
- 給水方式、計算条件、図面の読み方を判定軸として書く
- 数値計算問題でも、答えだけでなく「どの条件を拾うか」を示す

### 給水装置工事事務論
- 主任技術者が何をしなければならないかを書く
- 技能者配置、記録保存、講習、認証、法令遵守の違いを整理する

### 給水装置の概要
- 器具や材料の役割、用途、構造、設置位置を書く
- 故障問題では、原因と対策を対で書く

### 第8章（施工管理）
- 工程、品質、安全、道路復旧のどれが主題かを先に示す
- 安全管理では危険源、対策、責任主体を書く

## この試験で避けるべき悪い解説
- `選択肢2：正しい。`
- `記述のとおり。`
- `したがって答えは【3】`
- 法令や参考書の文章を長く引用するだけ
- 章をまたいだ無関係な知識を増やしすぎる
