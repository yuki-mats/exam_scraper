# 給水装置工事主任技術者試験 試験プロフィール

この文書は、給水装置工事主任技術者試験の章立て、出題傾向、問題形式を整理し、解説文作成や `category.json` 検討の共通前提とするための資料である。

## 試験全体の見取り図
- 参考書写真から、この試験は概ね次の8章構成で整理されている。
  1. 公衆衛生概論
  2. 水道行政
  3. 給水装置工事法
  4. 給水装置の構造及び性能
  5. 給水装置計画論
  6. 給水装置工事事務論
  7. 給水装置の概要
  8. 第8章（施工管理）
- 章扉や出題傾向表から、章ごとに出題数と最低必要得点数が整理されており、学習者は「全範囲を均等に覚える」よりも、**出題数が多く、基準・制度・数値・工程が絡む章を重点的に学ぶ**必要がある。

## 章ごとの性格
### 1. 公衆衛生概論
- 水道水の衛生、安全性、水質基準、浄水処理、塩素消毒、水系感染症、水質汚染物質を扱う。
- 出題は、原因物質、健康影響、浄水処理の目的、基準項目の意味を結び付けて問われやすい。

### 2. 水道行政
- 水道法の目的・定義、供給規程、給水装置、指定給水装置工事事業者制度、主任技術者制度、水質管理、水道事業者の業務などを扱う。
- 頻出のひっかけは、**主語の入替え**と**権限主体の取り違え**である。

### 3. 給水装置工事法
- 給水管の取出し・分岐、埋設・明示、水道メーター、配管・接合、現場管理、検査・維持管理を扱う。
- 問われ方は実務寄りで、**どこに設置するか、どう施工するか、何を確認するか**が中心になる。

### 4. 給水装置の構造及び性能
- 構造・材質基準、各種性能基準、逆流防止、クロスコネクション、システム基準、水の汚染防止を扱う。
- 重要度が高く、**性能基準の名前と中身の対応**、**システム基準と個別性能基準の違い**を整理して理解する必要がある。

### 5. 給水装置計画論
- 基本調査、給水方式、管径決定、水理計算、受水槽計画、図面読解を扱う。
- 図面読解と計算が絡み、**どの値を使うか、どの順で判断するか**が重要になる。

### 6. 給水装置工事事務論
- 主任技術者の職務、技能者の位置付け、記録、関係法令、建設業法、労働安全衛生を扱う。
- 暗記に見えて、実際は**誰が何をする義務を負うか**、**どの条件で許可や届出が必要か**が問われる。

### 7. 給水装置の概要
- 配管材料、継手、弁、水栓、水道メーター、故障と対策を扱う。
- 名称問題に見えても、**用途、構造、設置位置、故障原因**まで聞かれることがある。

### 8. 第8章（施工管理）
- 工程管理、品質管理、安全管理、道路復旧、施工計画などを扱う。
- **工程と品質の区別**、**安全管理の責任主体**、**道路・公衆災害対策**が重要になる。

## 問題形式の特徴
- 単純な一問一答ではなく、次の形式が多い。
  - 正誤判定型
  - 正誤の組合せ型
  - 語句補充型
  - 数値計算型
  - 図面・系統図読解型
- このため、解説では正答だけでなく、**どの判断軸で他の選択肢を落とすか**まで示すと学習効果が高い。

## 頻出の判定軸
- 主語の違い:
  - 水道事業者
  - 指定給水装置工事事業者
  - 給水装置工事主任技術者
  - 技能を有する者
  - 道路管理者
  - 警察署長
  - 埋設物管理者
- 適用範囲の違い:
  - どの器具・設備・工事に基準がかかるか
  - 直結式か受水槽式か
  - 最終の止水機構の流出側かどうか
- 数値基準:
  - 圧力
  - 温度
  - 幅員
  - 深さ
  - 口径
  - 距離
- 工程順序:
  - 調査 → 設計 → 協議 → 施工 → 検査
  - 取水 → 導水 → 浄水 → 送水 → 配水

## 解説文作成・カテゴリ設計への示唆
- 解説文では、主語・数値・順序・適用範囲を優先して説明すると、この試験らしい学習価値が出る。
- `category.json` では、章名だけでなく、**復習単位として意味がある論点粒度**に切る必要がある。
