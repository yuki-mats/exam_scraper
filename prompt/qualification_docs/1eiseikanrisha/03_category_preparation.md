# 第一種衛生管理者 category.json 準備メモ

## 根拠

- 公式分類は、公益財団法人 安全衛生技術試験協会「第一種・第二種衛生管理者の紹介」の試験科目・範囲を基準にする。
- 第一種衛生管理者は、労働衛生と関係法令がそれぞれ「有害業務に係るもの」と「有害業務に係るもの以外のもの」に分かれ、労働生理は単独科目として出題される。
- `output/1eiseikanrisha/questions_json/20031` から `20054` までの `00_source` で、平成26年10月公表から令和8年4月公表までの出題ラベルと問題本文を確認した。
- 法令論点の参照範囲は `01_law_reference_policy.md` に従う。

## folder 設計

公式の得点範囲をそのまま5つの folder にする。

1. `01_関係法令：有害業務`
2. `02_労働衛生：有害業務`
3. `03_関係法令：一般`
4. `04_労働衛生：一般`
5. `05_労働生理`

この資格では、公式上位分類をさらに複数 folder に分割しない。各 folder の下位に7前後の questionSet を置けば、一覧で復習範囲を把握できるためである。公式得点範囲への対応は `category.json` の `metadata.officialCategoryMappings` に1対1で残す。

## questionSet 境界

- 有害業務に係る法令問題は、法令上の義務・資格・設備・測定・特別規則・労働基準法上の就業制限で切る。
- 有害業務に係る労働衛生問題は、健康障害、ばく露、作業環境改善、保護具、特殊健康診断で切る。法令上の測定義務は法令側、測定結果の評価や改善手段は労働衛生側へ置く。
- 一般の関係法令問題は、管理体制、衛生委員会・教育、健康診断・面接指導、事務所衛生基準、労働時間・休暇、年少者・女性・妊産婦、受動喫煙防止で切る。
- 一般の労働衛生問題は、作業環境、健康保持増進・メンタルヘルス、疾病予防、救急処置、感染症・食中毒、統計で切る。
- 労働生理問題は、人体の器官系と機能で切る。法令上のストレスチェックは関係法令へ、睡眠・ストレス反応そのものは労働生理へ置く。

## 迷う境界

- 産業医・衛生管理者は、有害業務を含む事業場の専属・専任要件が主題なら `1-1 衛生管理体制`、一般的な選任や職務が主題なら `3-1 一般の安全衛生管理体制` とする。
- 作業主任者や免許・技能講習は、法令上の選任・資格要件が主題なら `1-2 作業主任者・資格・特別教育` とする。作業による健康障害が主題なら労働衛生側へ置く。
- 作業環境測定は、測定対象・頻度・作業環境測定士・記録保存を問う場合は `1-4 作業環境測定`、測定結果の評価や改善措置を問う場合は `2-6 作業環境改善と保護具` とする。
- 騒音、振動、放射線、高圧、温熱などは、有害業務の健康障害なら `2-4 物理的有害因子`、一般事務環境の温熱・換気・照明なら `4-1 一般作業環境` とする。
- メンタルヘルスは、指針に基づく体制や実施手続なら `4-2 健康保持増進・メンタルヘルス`、法令上のストレスチェック実施者や義務なら `3-2 衛生委員会・教育` とする。
- 受動喫煙は、健康増進法や施設管理者の義務を問う場合は `3-7 受動喫煙防止`、作業環境の換気計算だけを問う場合は `3-4 事務所衛生基準` 又は `4-1 一般作業環境` とする。

## 04工程への注意

03cでは問題側の `questionSetId` は付与しない。04工程では、`examLabel` だけで機械的に決めず、問題文と選択肢が問う復習論点を一問ずつ確認して紐付ける。
