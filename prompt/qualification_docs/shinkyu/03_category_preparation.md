# 鍼灸師 `category.json` 準備メモ

この文書は、鍼灸師過去問の `questionSetId` 紐付けを、公式出題基準に沿った復習単元として保つための補助資料である。

## 分類の正本

- 共通方針は [category_taxonomy_policy.md](/Users/yuki/development/exam_scraper/prompt/qualification_docs/category_taxonomy_policy.md) を正本とする。
- 鍼灸師固有の分類は、AHAKI の公式出題基準 2026年（令和8年）版を正本とする。
- `output/shinkyu/category/category.json` は、試験科目を `folders`、大項目を `questionSets` にしたものである。

## 基本原則

- `questionSetId` は、元サイトの `category` が `午前問題` / `午後問題` かどうかでは決めない。
- 受験者がその設問を復習するとき、公式出題基準のどの大項目へ戻るべきかで決める。
- `questionSetId` に使ってよいのは、`category.json` の `questionSets[].questionSetId` のみである。
- `folders[].folderId` を設問へ直接付与しない。

## 境界ルール

- 解剖学と臨床医学で迷う場合、設問が構造・局在・支配を問うなら解剖学、疾患・症候・検査・治療判断を問うなら臨床医学へ寄せる。
- 生理学と病理学で迷う場合、正常機能なら生理学、異常・病因・炎症・腫瘍などなら病理学へ寄せる。
- 臨床医学総論と各論で迷う場合、診察法・検査法・症候の一般論なら総論、個別疾患なら各論へ寄せる。
- 東洋医学概論と東洋医学臨床論で迷う場合、概念・診断理論・証の立て方なら概論、症候への治療適用なら臨床論へ寄せる。
- 経絡経穴概論と東洋医学臨床論で迷う場合、経穴名・所属・部位・要穴分類が主題なら経絡経穴概論、症候に対する選穴や治療方針が主題なら臨床論へ寄せる。
- はり理論ときゅう理論は、施術手技・刺激・リスク管理・治効理論が、鍼か灸かで明確に分ける。
- 関係法規と医療概論で迷う場合、法令上の免許・業務・罰則・関係法規は関係法規、医療制度・倫理・社会保障は医療概論へ寄せる。

## category.json を更新する場合

通常は更新しない。現行の `category.json` は公式出題基準ベースの正本である。

更新が必要な場合は、次を満たす。

- 公式出題基準の改訂、またはユーザー指定の専門家資料がある。
- 既存 `questionSets` の説明・`matchingHints` 改善では対応できない理由がある。
- 追加・削除・名称変更後に、5,560問の `questionSetId` を再検証する。
