# 鍼灸師 法令参照方針

この文書は、鍼灸師過去問の `explanationText` で法令・制度問題を扱う際の参照範囲を定める。

## 基本方針

- 関係法規、医療概論、衛生学・公衆衛生学の一部では、法令・制度上の根拠確認が必要になる。
- 法令条文が正誤判断に直接関わる場合は、`isLawRelated=true`、`lawGroundedExplanationNotNeeded=false` にする。
- 医学知識や東洋医学知識だけで判断できる問題では、無理に法令参照を作らない。
- 現行法と出題当時法令を突き合わせ、現行法で正誤が明らかに変わる場合は現行法ベースへ `correctChoiceText` / `explanationText` を更新する。
- 更新した場合の保存先と補足質問の採否は、上位READMEが示す共通field契約と工程03に従う。鍼灸師だけの補足質問必須ルールは設けない。

## 優先して確認する法令・制度

- あん摩マッサージ指圧師、はり師、きゅう師等に関する法律
  - 免許、業務、施術所、広告、届出、罰則、守秘義務に関する問題で確認する。
- あん摩マッサージ指圧師、はり師、きゅう師等に関する法律施行令
  - 試験、免許、行政手続などで必要な場合に確認する。
- あん摩マッサージ指圧師、はり師、きゅう師等に関する法律施行規則
  - 免許申請、名簿、施術所届出、広告事項などの細部で確認する。
- 医療法
  - 医療提供体制、医療機関、広告、医療安全などが直接問われる場合に確認する。
- 医師法、保健師助産師看護師法、理学療法士及び作業療法士法
  - 他職種の業務範囲や資格制度と比較する問題で確認する。
- 健康保険法、国民健康保険法、高齢者の医療の確保に関する法律、介護保険法
  - 医療保険、国民医療費、高齢者医療、介護保険制度が正誤判断に関わる場合に確認する。
- 感染症の予防及び感染症の患者に対する医療に関する法律、予防接種法
  - 感染症分類、届出、予防接種制度が直接問われる場合に確認する。
- 地域保健法、母子保健法、学校保健安全法、労働安全衛生法
  - 保健制度、母子保健、学校保健、産業保健が主題の場合に確認する。

## lawReferences を作る条件

- 選択肢の正誤判断に、免許、業務範囲、届出、広告、罰則、制度上の義務や定義が直接関わる。
- 解説本文で法令名や条項を明記することで、受験者が判断根拠を確認できる。
- 出題当時と現行法の差分が、過去問の理解に影響する。

## lawReferences を作らない条件

- 解剖、生理、病理、臨床医学、東洋医学、経絡経穴、はり理論、きゅう理論など、医学・施術知識だけで正誤判断が完結する。
- 法令名が背景に出ていても、選択肢の正誤は統計、疫学、医学知識、一般制度理解で説明できる。
- 条文番号や法令IDを一次情報で確認できない。

## 迷った場合

- `isLawRelated=true`、`lawGroundedExplanationNotNeeded=false` に倒す。
- `lawReferences` は推測で入れず、条文を一次情報で確認できた場合だけ作る。
- スコープ外の法令が必要になった場合は、問題文または選択肢上の根拠を確認し、この文書へ追記してから使う。
