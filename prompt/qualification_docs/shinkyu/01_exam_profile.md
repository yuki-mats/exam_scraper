# 鍼灸師 試験プロフィール

この文書は、鍼灸師過去問の `03_prompt_add_explanationText.md` と `04_prompt_link_questionSetId.md` の共通前提である。

## 試験全体の見取り図

- 対象は、はり師・きゅう師国家試験の取得済み過去問である。
- `output/shinkyu/questions_json` には、1993〜2026年の34年分、合計5,560問がある。
- 1993〜2020年は各160問、2021〜2026年は各180問である。
- 現在の `00_source` は全問 `true_false`、選択肢は全問4肢である。
- `questionIntent` は `select_correct` が多いが、`select_incorrect` も約700問あるため、設問末尾の「正しい」「誤っている」「でない」を必ず確認する。

## 科目構成

`output/shinkyu/category/category.json` は、AHAKI の公式出題基準 2026年（令和8年）版をもとに、試験科目を `folders`、大項目を `questionSets` として構成している。

主な領域は次の通り。

- 専門基礎分野: 解剖学、生理学、病理学概論、臨床医学総論、臨床医学各論、リハビリテーション医学、衛生学・公衆衛生学、関係法規、医療概論
- 専門分野: 東洋医学概論、経絡経穴概論、はり理論、きゅう理論、東洋医学臨床論

## 出題の性格

- 西洋医学系科目では、用語の暗記だけでなく、疾患・症候・検査・治療・リハビリテーションの対応関係が問われる。
- 東洋医学系科目では、陰陽・五行・気血津液・臓腑・経絡経穴・証・治療原則の対応を混同しやすい。
- はり理論・きゅう理論では、施術方法、禁忌、リスク管理、治効機序、安全対策をセットで理解する必要がある。
- 関係法規と医療概論では、制度名や法令上の義務・免許・業務範囲を曖昧に扱うと誤学習につながる。

## 01〜04作業への示唆

- 01 `questionType`: 現在は全問 `true_false` だが、選択肢構造と設問文を目視で確認する。
- 02 `questionIntent`: 設問末尾を必ず確認し、正しいものを選ぶのか、誤っているものを選ぶのかを確定する。
- 03 `explanationText`: 各選択肢について、正誤だけでなく「どこで判断するか」を書く。
- 04 `questionSetId`: 午前/午後だけでは分類せず、公式出題基準の大項目へ復習先として紐付ける。
