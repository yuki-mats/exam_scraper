# 介護福祉士試験 `category.json` 準備メモ

この文書は、介護福祉士試験の `category.json` を設計・見直しする際に、どの粒度で論点を分けると復習しやすいかを整理するためのメモである。

## 大分類の基本方針
- まずは現行科目群を土台にする。
- ただし、学習者の復習単位として大きすぎる科目は、中論点へ分解する。

## 分けやすい中論点の例
### 社会の理解
- 介護保険制度
- 障害福祉制度
- 社会保障・社会福祉制度
- 家族・地域・権利擁護

### 介護の基本 / コミュニケーション技術
- 尊厳保持と自立支援
- 権利擁護
- 受容・共感・自己覚知
- 初期対応と信頼関係形成
- 多職種連携

### 生活支援技術
- 移動・移乗
- 食事
- 排泄
- 入浴・清潔
- 整容・更衣
- 住環境整備
- 福祉用具

### 介護過程
- アセスメント
- 目標設定
- 計画立案
- 実施と評価

### 発達と老化 / 認知症 / 障害 / こころとからだ / 医療的ケア
- 加齢変化
- 認知症の症状と支援
- 障害特性と支援
- 解剖生理
- 疾患と観察
- 医療的ケア

## 旧課程の扱い
- 2008年から2010年の旧科目は、現行の近い大分類へ寄せて扱う。
- 旧科目名をそのまま独立カテゴリにするより、現行の復習単位へ吸収した方が検索・復習の一貫性が高い。

## 解説文との整合
- `explanationText` で繰り返し出る判断軸は、カテゴリにも反映すると復習しやすい。
- 例えば次のような軸は、科目名よりも学習単位として有効な場合がある。
  - 本人意思の尊重
  - 制度の主体・対象
  - 認知症ケア
  - 自立支援と安全配慮
  - 観察と異常時対応
