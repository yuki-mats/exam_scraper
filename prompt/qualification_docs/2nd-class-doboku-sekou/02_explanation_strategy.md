# 二級土木施工管理技士試験 解説文戦略

この文書は、`03_prompt_add_explanationText.md` で二級土木施工管理技士試験の `explanationText` を書く際に、何をプラスアルファとして補うべきかを整理した補助資料である。

## この試験の解説で最優先にすべきこと
- `正しい。` `間違い。` だけで終わらせず、どの判断軸でそう言えるのかを示す。
- 特に次のどれで判定したかを明示すると学習効果が高い。
  - 工法・材料・試験の対応
  - 適用場面
  - 数値条件
  - 施工順序
  - 主体や管理者
  - 図中部材の位置関係

## 頻出のひっかけ
- 似た工法名の取り違え
- 試験名と、そこから求められる値・性質の入替え
- コンクリート施工での時間、層厚、締固め条件の混同
- 舗装工法の名称と適用場面の混同
- 河川・港湾・構造物で、部材名称と役割の混同
- 労働基準法、労働安全衛生法、建設業法などの主語の混同
- 許可、届出、選任、教育の主体と時点の取り違え

## 正誤の組合せ型での解説方針
- この試験では、本文に説明があり、`choiceTextList` が組合せだけになる問題がある。
- この型では、学習者が知りたいのは「どの組合せが正しいか」ではなく、「元の説明のどこが誤りか」である。
- したがって、誤答肢の解説は組合せ番号の違いだけで終わらせず、差が出る本文の語句・数値・条件を明示する。
- 正答肢でも、成立理由を最低1つは言語化する。`正解だから正しい` という書き方は避ける。

## 図表問題での解説方針
- `flash_card` や `group_choice` では、図を見なくても判定軸が残る説明にする。
- 例:
  - `根固工は堤防や護岸の基部洗掘を防ぐため、法先や基礎付近に設ける。`
  - `曲げモーメント図は荷重点と支点条件から形状を決める。`
- 図中位置だけでなく、その部材や図形が何を示しているかまで書く。

## 解説の基本形
- 1文目:
  - `正しい。` または `間違い。`
- 2文目:
  - 誤りの箇所、または成立理由を一言で述べる
- 3文目以降:
  - 類題でも使える補足知識を入れる

例:
```json
"間違い。\n\n誤っているのは、スランプ試験を空気量測定の試験としている点である。スランプ試験はフレッシュコンクリートの軟らかさをみる試験であり、空気量は別の試験で確認する。"
```

## 章別の補足知識
### 工学基礎知識
- 試験名と求められる値を対で書く。
- 力学・水理では、式の使いどころと前提条件を簡潔に補う。

### 土工・基礎工・構造物
- 工法の適用場面、目的、地盤条件を対で書く。
- 類似工法との違いを1点だけでも添える。

### コンクリート
- 材料、配合、施工、試験のどの論点かを最初に示す。
- `施工` 問題では、時間・層厚・締固め・養生のどれで判定したかを明示する。

### 河川・道路舗装・ダム・トンネル・海岸港湾
- どの施設や工法の問題かを先に特定する。
- 部材名称だけでなく、役割や適用場面まで説明する。

### 法令・契約・施工管理
- 主語、義務、要件、禁止事項、届出先を明示する。
- 他法令と混ざりやすい論点では、何法の話かを先に書く。

## この試験で避けるべき悪い解説
- `選択肢2が正しい。`
- `記述のとおり。`
- `したがって答えは1である。`
- 図を見ないと何も残らない説明
- 法令や基準文を長く引用するだけの説明
