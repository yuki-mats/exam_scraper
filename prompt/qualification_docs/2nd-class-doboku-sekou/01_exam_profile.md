# 二級土木施工管理技士試験 試験プロフィール

この文書は、二級土木施工管理技士試験の章立て、出題傾向、問題形式を整理し、解説文作成や `category.json` 検討の共通前提とするための資料である。

## 年度による形式差
- 平成29年度から令和5年度までは、`土木`、`薬液注入`、`鋼構造物塗装` の区分が `examLabel` に現れる回がある。
- 令和6年度前期と令和7年度前期は、`土木（ユニットA〜E）` のようにユニット分割された形式が現れる。
- 令和6年度以降は `施工管理法（基礎的な能力）` を独立した分類として扱う必要があり、従来の施工計画・工程・安全・品質と問題の見え方が近くても、復習単元としては分けて考える。

## 問題形式の特徴
- 取得済み 1,376問の大半は `true_false` で、設問文に `適当なもの` / `適当でないもの` が明示される。
- 一部に `flash_card` があり、図中部材名、舗装形式、構造名称、部材の読み取りを問う。
- 近年回には `group_choice` があり、式や図そのものを選ばせる問題が混ざる。選択肢テキストが空でも `question_images` が根拠になる。
- 組合せ問題では、単なる語句知識ではなく、組み合わせた結果としてどこが誤っているかを見抜く力が必要になる。

## 頻出の判定軸
- 主題の見分け:
  - 材料・工法・試験・施工手順のどれを問うか
  - 現場作業の安全か、法令上の義務か、品質管理か
- 数値や条件:
  - 締固め、養生、施工時間、配合、試験条件
  - 許可・届出主体、規制対象、就業制限、作業主任者選任条件
- 用語の対応:
  - 工法名と適用場面
  - 劣化機構と要因
  - 試験名と求められる情報
  - 施設・構造名称と図中位置

## 解説文作成・カテゴリ設計への示唆
- 解説文では、正誤だけでなく「なぜその工法・試験・法令になるのか」を一段具体化すると、この試験らしい学習価値が出る。
- `questionSetId` は参考書の章立てをそのまま写すだけでは足りず、学習者が誤答後に復習先を即決できる粒度で切る必要がある。
- `施工管理法（基礎的な能力）` は他章の再掲に見える問題でも、近年の出題形式では独立単元として扱う。
