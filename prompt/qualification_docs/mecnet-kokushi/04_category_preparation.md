# 医師国家試験 `questionSetId` 整備メモ

この文書は、医師国家試験の `questionSetId` を `category.json` のブループリント名へ寄せる際の補助メモである。

## 前提

- `output/mecnet-kokushi/category/category.json` は MHLW ブループリント命名を正本にしている
- 日常の紐付けは `category.json` の `questionSetName` と `matchingHints` を主根拠にする
- ここでは、迷いやすい境界だけを補足する

## 基本原則

- まず「何の論点を復習させたい設問か」で決める
- 病名が明確なら、総論よりも各論の具体カテゴリを優先する
- 症例問題でも、診断手技ではなく病態理解が主題なら疾患カテゴリへ寄せる
- 救急症例は、主題が救急初療なら救急カテゴリ、特定疾患の診断・治療なら疾患カテゴリを優先する
- 妊娠・分娩・新生児は、母体管理、分娩進行、新生児管理のどこが問われているかで切る

## 迷いやすい境界

- 感染症対策と感染症各論
  - 届出、隔離、院内感染対策、保健所対応は公衆衛生・感染症対策
  - 病原体、診断、治療、合併症は各論の感染症カテゴリ
- 救急と各臓器疾患
  - BLS / ACLS / ショック初療 / トリアージは救急
  - 心筋梗塞、脳卒中、急性腹症の病態や診断は各臓器カテゴリ
- 公衆衛生統計と疫学計算
  - 指標の定義、SMR、感度・特異度、罹患率・有病率は公衆衛生統計
  - 制度の実施主体や保健所業務は制度寄りカテゴリ
- 母体保健と産婦人科各論
  - 母子健康手帳や健診制度は法令・制度
  - 妊娠、分娩、産褥、新生児管理は産婦人科・小児科寄り

## 運用

- 紐付けに迷う問題が繰り返し出る場合は、まず `category.json` の `matchingHints` を改善する
- この文書だけを根拠に新しい `questionSetId` を増やさない
