# 医師国家試験 `lawReferences` 対象法令スコープ

この文書は、医師国家試験で `lawReferences` を作る際に、優先して確認する法令スコープを整理したものである。

まずこの表の範囲から確認し、e-Gov の全法令を無差別に探さない。ここにない法令を使う場合は、問題文・選択肢・解説候補にその法令が直接関係する根拠を確認してから追加する。

## スコープ外法令を追加する条件

次を全て満たす場合だけ追加する。

- 問題文・設問文・選択肢・解説候補のいずれかに、その法令が正誤根拠になる合理的根拠がある
- 一次情報で正式法令名、`lawId`、条番号を確認している
- この文書へ追記し、使う場面を明示する

満たせない場合は、推測で `lawReferences` を作らない。

## 補足

- `lawId` は `lawReferences` 付与時に e-Gov の現行法令ページで最終確認する。
- `母体保護法`、`臓器の移植に関する法律`、`死体解剖保存法` は医師国家試験では頻度が高くないため `conditional` とするが、出題された場合の学習価値は高い。
